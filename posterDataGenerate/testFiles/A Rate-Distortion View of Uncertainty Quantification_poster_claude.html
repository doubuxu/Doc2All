Our Method: Distance-Aware Bottleneck (DAB)

1> Treat the training dataset as a source generating training examples.

$^ { 2 > }$ Map each datapoint to a distribution through a stochastic encoder.

3> The codebook contains the centroids of the encoders in terms of a statistical distance. Uncertainty is quantified by the expected distance from the codebook: uncertainty $\cdot ( x _ { \mathrm { t e s t } } ) = \mathbb { E } [ D ( p ( z \mid x _ { \mathrm { t e s t } } ; \theta ) , q _ { \kappa } ( z ; \phi ) ) ]$
