Introduction

Deep Graph Learning (DGL) has emerged as a crucial technique across various domains. However, recent studies have exposed vulnerabilities in DGL models, such as susceptibility to graph injection attacks (GIAs). Existing research only provides sample-wise certificates by verifying each node independently, leading to very limited certifying performance. In this paper, we present the first collective certificate against GIAs, which certifies a set of target nodes simultaneously.

Randomized Smoothing

For any node classifier f(·), its smoothed classifier g(·) is created by:

  • 1. Node-aware Bi-Smoothing (Y. Lai. et. al. S&P2024): φ(G) = (φe(G), φn(G))
  • 2. Majority vote: gv(G) := arg max P(fv(φ(G)) = y)

The goal of certified robustness is to verify: gv(G) = gv(G'), ∀ G' ∈ Bρ,τ(G).

Collective Certificate Problem

Bρ,τ(G): the attacker can inject ρ malicious nodes, with τ malicious edges per node.

T: A set of target nodes.

min |T| - Σ I{gv(G') ≠ gv(G)}

s.t. |Ṽ| ≤ ρ, δ(ṽ) ≤ τ, ∀ ṽ ∈ Ṽ

Let p(Ev) denotes the probability that the malicious messages is NOT intercepted, then we have that the absolute change of prediction probability is bounded by p(Ev):

|pv,y(G) - pv,y(G')| ≤ p(Ev)

Collective Certificate via Optimization

p(Ev): the probability that the malicious messages is not intercepted.

cv := pv,y*(G) - max pv,y(G).

max Σ I{p(Ev) ≥ cv/2}

s.t. |Ṽ| ≤ ρ, δ(ṽ) ≤ τ, ∀ ṽ ∈ Ṽ

The remaining |T| - M* nodes are certified robust.

Sample-wise v.s. Collective Certificate

Sample-wise certificates essentially verify the prediction for a node one by one, assuming that the attacker can craft a different perturbed graph each time to attack every single node (Figure 1, top).

However, in reality, the attacker can only produce a single perturbed graph to simultaneously disrupt all predictions for a set of target nodes (Figure 1, bottom).

Evaluation Results

With the same smoothing parameter, both proposed collective certificates achieve a higher certifiable radius (Figure 3), outperforming the sample-wise approach significantly when the ρ is large. For example, in the Citeseer dataset, when ρ = 140, our Collective-LP1 and Collective-LP2 have the certified ratios of 73.0%, and 81.2%, while sample-wise can certify 0.0% nodes.
