Conformal Semantic Keypoint Detection with Statistical Guarantees

Heng Yang, Marco Pavone

Workshop on Robot Learning: Trustworthy Robotics

Object Pose Estimation: why untrustworthy?

Current pipelines rely on heuristic prediction which are not guaranteed to be close to groundtruth. Popular heuristics (e.g., RANSAC) get trapped in local minima and fail without notice.

Our method: conformal keypoint detection

Paradigm shift: from heuristic prediction to probabilistically correct prediction sets.

Key ingredient: inductive conformal prediction (ICP) — finite-sample uncertainty quantification without assumptions on the data distribution or the prediction model.

  • Requirement #1: Calibration set
  • Requirement #2: Nonconformity function
  • Guarantee: Cover the true keypoints with a user-specified probability (e.g., 90%)

Experiments: works out of the box!

Setup: 200 images for calibration, 1214 images for testing.

User: give me 90% and 95% coverage of the true keypoints.

The results demonstrate valid coverage (90%, 95%) and small size of the prediction sets across various objects in the LineMOD Occluded dataset.
