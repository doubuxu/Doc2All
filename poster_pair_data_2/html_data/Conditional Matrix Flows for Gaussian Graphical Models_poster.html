Gaussian Graphical Models: Graphical Lasso

  • GGMs provide structural information: Ωij = 0 implies conditional independence of features i and j given all remaining features
  • Maximum likelihood estimate of Ω is not sparse and for d > n is not unique → penalized likelihood to encourage sparse Ω

argmaxΩ>0 {log det Ω - Tr(1/n SΩ) - λ||Ω||1}

Intractable because the objective is highly non-convex

Graphical Lasso: use convex relaxation ||Ω||0 → ||Ω||1

Benefits

  • Solution path: frequentist approach provides solutions as a function of λ

Limitations

  • Over-shrinking: l1 norm causes well-known over-shrinking of features
  • No credibility interval

Bayesian Graphical Lasso

  • Bayesian approach: p(Ω|S, λ) ∝ p(S|Ω) p(Ω|λ)
  • Encourage sparsity with double exponential prior (DE)

p(S|Ω) ∝ Wd(n, Ω⁻¹)
p(Ω|λ) ∝ Wd(d + 1, λ1d) ∏ DE(ωij|λ) I[Ω > 0]

Benefits

  • Credibility interval: can be computed within Bayesian framework

Limitations

  • Expensive posterior sampling: needs Gibbs sampler
  • Over-shrinking: prior cannot be generalized to sub-l1 pseudo-norms
  • Expensive solution path: independent Markov chains for each λ value

Conditional Matrix Flow for Gaussian Graphical Models

Goal: unify benefits of frequentist and Bayesian frameworks while overcoming limitations

We propose a general framework for variational inference with matrix-variate Normalizing Flows in GGMs. As a key improvement on previous work, we train with one flow a continuum of sparse regression models jointly for all regularization parameters λ and all lq norms, including non-convex sub-l1 pseudo-norms.

VARIATIONAL INFERENCE
  • Posterior approximation and efficient sampling via Normalizing Flow
SUB-lq PSEUDO-NORMS
  • Generalized Gaussian Distributions (GGD) as sparsity-inducing priors
CONDITIONAL FLOW
  • Condition the Normalizing Flow on λ and q to obtain solution paths

ARCHITECTURE

Combine benefits

  • Credibility intervals: sample-based
  • Solution paths: recover Bayesian GL

Overcome limitations

  • Cheap posterior sampling: no Gibbs
  • No over-shrinking: GGD prior
