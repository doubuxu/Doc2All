Motivation

Our objective is to fully disentangle 3D shape, viewpoint, object and background generation on photorealistic synthesis.

Contribution

Photorealistic modelling with a 3D morphable model from 2D image supervision. State-of-the-art disentangled control over shape, viewpoint, object and scene appearance.

  • 1. Introducing Template Object Coordinates (TOCS) as a powerful deformation-equivariant descriptor of 3D shape.
  • 2. Conditioning a 2D style-based DNR with TOCS map for shape and appearance disentanglement.

Overall Pipeline

Morphable Renderer

A SIREN network learns to offset world points to template space. This allows to express each instance as a deformation of an instance-agnostic template shape.

The TOCS Map captures all and only the 3D geometrical aspects of image formation.

Deferred Neural Renderer

The 2D TOCS map conditions StyleGan2 blocks for the object appearance synthesis. Foreground and background are generated independently and composed for foreground/background disentanglement.
