Contributions

  • A generic framework to design a GNN from any fragment of an algebraic language.
  • The instantiation of the framework on ML(L3) resulting in G2N2, a provably 3-WL GNN.
  • Experiments demonstrating that G2N2 outperforms existing 3-WL GNNs on various downstream tasks.

MATLANG and WL

Definition 1: MATLANG is a matrix language with operation set {+, ·, ⊙, T, Tr, diag, 1, ×, f}.

Restricting operations to a subset L defines a fragment of MATLANG ML(L).

Context-Free Grammar

Definition 2: A Context-Free Grammar (CFG) G = (V, Σ, R, S) with a finite set of variables.

Grammar of literature GNNs
