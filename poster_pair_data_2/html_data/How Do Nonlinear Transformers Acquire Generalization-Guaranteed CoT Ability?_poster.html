How Do Nonlinear Transformers Acquire Generalization-Guaranteed CoT Ability?

Hongkang Li¹, Meng Wang¹, Songtao Lu², Xiaodong Cui², Pin-Yu Chen²

1: Rensselaer Polytechnic Institute; 2: IBM Research
github.com/research-project-cot

Overview

  • Theoretically characterize how training Transformers can enable chain-of-thought (CoT).
  • Quantitative analysis of how context examples affect CoT performance.
  • Theoretical study of why CoT outperforms ICL.

Background and motivation

CoT augments K-step reasoning examples to the query for generation, i.e., an extension of in-context learning (ICL) by multi-step examples.

  • Existing works [1,2,3]: the expressive power.
  • The problem of why a Transformer can be trained to conduct CoT is less investigated.

Problem formulation

Study learning K-step reasoning tasks. Each task f is a composition of {fᵢ} and outputs {zᵢ} for the input z₀.

Learning model: one-layer Transformer F(P) = Σ Wᵥ p̃ᵢ softmax( (Wₖ p̃ᵢ)ᵀ W_Q p̃_query ), where P is the input prompt.

Training: Following theoretical works [4,5] for ICL, we aim to solve an empirical risk minimization by stochastic gradient descent with squared loss.

CoT Inference: The testing prompt consists of examples and a query. CoT repeats two steps for K steps:

  • Greedy decoding to generate the next step.
  • Add the generation to the end of the prompt.

ICL Inference: The testing examples only contain the input and the final-step output.

Theoretical results

Data & Tasks: The training and testing data are formulated by TTR/TSR patterns. Consider erroneous testing prompts: reasoning in some examples contains incorrect steps.

Theorem 1 (Convergence): With the number of context examples and iterations linear in Ω(α⁻¹), the population risk is ≤ O(ε).

Theorem 2 (CoT Generalization): To achieve a zero CoT error, the number of context examples for task f is proportional to (α' τᶠ ρᶠ)⁻².

Theorem 3 (Comparison with ICL): The success of ICL requires a dominant fraction of correct input-label examples (Condition 1), but CoT does not.

The mechanism of CoT
Proposition 1: The attention weights are concentrated on prompt columns with the same TSR pattern and step index.

Numerical experiments

  • Condition 1 is crucial for the success of ICL but not for CoT.
  • Training dynamics of CoT: same TSR & step matter.
