Kernel Ridge Regression

A (finite) rank-M kernel K is a map K: X x X -> R defined for any x, x' in X by K(x, x') = sum_{k=1}^M lambda_k psi_k(x) psi_k(x').

Fix a ridge lambda > 0, the regressor f_z,lambda of the finite-rank kernel K is the (unique) minimizer of the empirical risk functional.

Test Error

Note that {psi_k}_{k=1}^M are orthonormal functions. Write target function f* = sum_{k=1}^M gamma_k psi_k + f_perp.

The finite-rank KRR test error is decomposed into Bias-Variance: R_z,lambda = bias + variance.

Where bias measures the error when there is no noise in the label, and variance is defined to be the difference between test error and bias.
