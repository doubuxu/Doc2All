Preliminary

Sequential Decision Making

  • Markov Decision Process (MDP): {pt(St+1, Rt | At, St)}t
  • Stationarity assumption: H0: {pj(Sj+1, Rj | Sj, Aj)}0≤j≤T-1 are homogeneous over time j.
  • Offline reinforcement learning (RL): static dataset without further interactions.
  • Additional challenges in modern RL problems:
    • State variable St can be high-dimensional (e.g., 7056 for Atari).
    • Dynamics pt can be highly complex (e.g., autonomous driving).
  • Modern ML methods well-suited for non-linearity and high-dimensionality, however, need careful treatment for proper statistical inference.

Data Split

Maximum-type test statistic:

Γ = maxt, b [ t(T-t)/T2 σ(t, hb)-1 |1/|I2| ∑ ψi | ]

where ψi involves a plug-in term and an augmented term to achieve double robustness.

  • hb is a test function from a rich function class and B is the number of test functions.
  • Nuisance functions are estimated using the data from I1. One can use flexible machine/deep learning tools.
  • Gaussian multiplier bootstrap is used to approximate the limiting distribution and compute the p-value.

Theoretical Property

Double Robustness: Suppose H0 holds. For any h ∈ H and t ∈ (0, T-1), we have E[ψi] = 0. Moreover, the equation is doubly robust, i.e., for any estimated nuisance functions, the equality holds as long as either the transition model or the ratio model is correctly specified.

Size: Suppose B = O((NT)c) for any c > 0 and T = O(T) for any t. Under certain regularity conditions including strict stationarity and geometric ergodicity, the convergence speed of nuisance functions satisfies specific rates, then we have:

P(p ≤ α | H0) = α + o(1)

Application to Intern Health Study Data

  • Intern Health Study [2]: Wearable and survey data of 1565 medical interns in the U.S. collected for 26 weeks.
  • The goal is to improve health-related outcomes via mobile-push notifications.
  • We consider the next week's daily average step count as the reward Rt.
