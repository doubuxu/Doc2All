PatchRot: A Self-Supervised Technique for Training Vision Transformers

Sachin Chhabra, Prabal Bijoy Dutta, Hemanth Venkateswara, Baoxin Li

NeurIPS
github.com/patchrot-project

Problem Statement

  • Vision transformers require a huge amount of labeled data to outperform convolutional neural networks.
  • The features learned by such a network are highly transferable and can be used for similar tasks. However, labeling a dataset is generally expensive and time-consuming.
  • Self-supervised learning can alleviate this problem by learning rich features without the need for manual annotation of labeling the dataset.
  • Existing self-supervised techniques were designed to train ConvNets. However, in recent years, Vision Transformers (ViT) have surpassed ConvNets.
  • Existing self-supervised techniques can be applied to ViTs. However, ViTs process images differently than ConvNets.

PatchRot

  • ViT splits the image into a series of input patches.
  • We randomly rotate each patch by 0°, 90°, 180° or 270°.
  • In addition to rotating patches, we create 4 more versions of the image by rotating image by 0°, 90°, 180° or 270°.
  • For every input we use 5 versions of image as input. (Note: we do not rotate image and patches together).
  • We introduce new classification heads to the ViT.
  • We use the original head for classifying the image and the new heads are used to classify the rotation angle of the patches.
  • To ensure network does not use shortcuts to classify the angles, we use a buffer B for each patch i.e. extract patch P from a P+B region.
  • Buffer reduces the image size. The self-supervised training is performed at smaller size and then adjusted to the original size by linear interpolation while adapting to downstream task.

Conclusions

  • PatchRot trains the network to predict the rotation angle of the patches as well as of the image.
  • Our approach outperforms rotation prediction and supervised learning by a fair margin showcasing the superiority of the features learned.
  • PatchRot learns to attend to the object of the image with no supervised training.
