Overview

TL;DR: We learn an ensemble of neural networks that is diverse with respect to their input gradients.

Repulsive deep ensembles (RDEs) [1]

Description: Train an ensemble using Wasserstein gradient descent, which employs a kernelized repulsion term to diversify the particles.

  • The driving force directs particles towards high density regions.
  • The repulsion force pushes particles away to enforce diversity.

Problem: Weight-space or function-space repulsion often results in underfitting or is ineffective.

First-order repulsive deep ensembles (FoRDEs)

Possible advantages:

  • Each member is guaranteed to represent a different function.
  • Issues of weight- and function-space repulsion are avoided.
  • Encourages learning different features for robustness.

Main takeaways

  • 1. Input-gradient-space repulsion can perform better than weight- and function-space.
  • 2. Better corruption robustness can be achieved by configuring the repulsion kernel.

Tuning the lengthscales Σ

Each lengthscale is inversely proportional to the strength of the repulsion force.
