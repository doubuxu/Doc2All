Biological learning is different from machine learning in key ways

Biological systems operate with:

  • sparse data
  • lower/efficient energy use
  • minimal/variable training
  • trade-offs driving adaptation

Machine learning operates with:

  • large datasets
  • high energy use
  • extensive training
  • trade-offs causing failure

What biological mechanisms could help enhance ML?

Self-optimization (SO) model

The SO model is a Hopfield network with:

  • updating of weights based on a Hebbian learning rule
  • periodic resets of network states

Able to combine optimization and memory stabilization?

Example: learning rate as a function of energy

1. Weight update rule (α: learning rate; s_i, s_j: states of nodes i and j)

w_ij(t + 1) = w_ij(t) + dw_ij(t),    dw_ij = α s_i s_j

2. Learning rate α as a function of energy E:

α = { a / (ΔE + ε_0) if γ ≠ 1; α_0 if γ = 1 }

SO model results for map-coloring problem. From left to right, E=5, E=8, E=12, E=20 (only the two middle results are valid maps).

Implementing mechanisms used by living systems to adapt to their environment may help solve trade-offs inherent to machine learning.

Energy distributions at the end of learning for 1000 runs. A slight difference can be observed between γ=1 (learning rate α constant) and γ=0.1 (variable learning rate), but not obviously significant.
