Introduction

Spherical CNNs
  • Principled way of learning from spherical signals, using spherical convolutions computed spectrally.
  • Equivariant to 3D rotations.
  • Respects spherical topology, robust to sampling.

Method

  • Based on spin-weighted SCNNs (Esteves et al, NeurIPS'20).
  • Phase collapse activation: x₀ ← W₁x₀ + W₂|x| + b
  • Efficient spin-weighted spherical harmonics on TPUs.
  • Spectral batch normalization, pooling and residual connection.

Molecular property regression

  • Represent a molecule as a set of spheres per atom, regress a set of properties.
  • One feature channel per atom type.
  • Use directional attributes only (no radius).

Weather forecasting

  • Take a set of atmospheric variables like temperature, wind, humidity, and predict their future values.
  • WeatherBench (Rasp et al): predict directly 3 or 5 days ahead.
  • Keisler (arXiv'22): predict iteratively in 6h increments.
