NeuPL: Neural Population Learning

Siqi Liu, Luke Marris, Daniel Hennes, Josh Merel, Nicolas Heess, Thore Graepel

Background

+ =
Transitive Skills
Strategic Cycles
Game-of-Skills

The policy space of symmetric zero-sum Game-of-Skills is composed of transitive skill dimensions and intransitive strategic cycles.

Solving intransitive strategy games: Game-theoretic population learning algorithms (e.g. FP, PSRO) offer convergence guarantees to an NE, where each policy best-responds to a mixture over predecessors.

Fictitious Play
PSRO-NASH

However, when combined with modern deep RL techniques, population learning comes at significant costs, for two reasons:

  • Lack of positive transfer across policies;
  • Suboptimal "good"-responses populating the policy population;

Can we leverage the expressiveness of modern deep NNs to enable a single model to explore, represent and reason about diverse strategies while retaining game theoretic guarantees using the same infrastructure as "self-play"?

Methods

Algorithm 1 Neural Population Learning (Ours)
1: Πθ(·|s, σ) ▷ Conditional neural population net.
2: Σ := {σi}i=1N ▷ Initial interaction graph.
3: F : RN×N → RN×N ▷ Meta-graph solver.
4: while true do
5: ΠθΣ ← {Πθ(·|s, σi)}i=1N ▷ Neural population.
6: for σi ∈ UNIQUE(Σ) do
7: Πθσi ← Πθ(·|s, σi)
8: Πθσi ← ABR(Πθσi, σi, ΠθΣ) ▷ Self-play.
9: U ← EVAL(ΠθΣ) ▷ (Optional) if F adaptive.
10: Σ ← F(U) ▷ (Optional) if F adaptive.
11: return Πθ, Σ
