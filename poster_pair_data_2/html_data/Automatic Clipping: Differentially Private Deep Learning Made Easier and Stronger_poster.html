ABSTRACT

Per-example gradient clipping is a key algorithmic step that enables practical differential privacy (DP) training for deep learning models. The choice of clipping threshold R is shown to be vital for achieving high accuracy under DP but difficult to tune.

We propose an easy-to-use replacement, called Automatic Clipping, that eliminates the need to tune R for any DP optimizers.

CONTRIBUTIONS

  • We propose a new per-sample gradient clipping function that does not require the clipping threshold R.
  • We give a convergence analysis of automatic DP-SGD in the non-convex setting.
  • We demonstrate on extensive language and vision tasks that automatic clipping outperforms or matches state-of-the-art.
  • We demonstrate minimal changes to adopt our automatic clipping in existing codebases.

R-INDEPENDENCE OF AUTOMATIC CLIPPING

We show that using automatic clipping, it suffices to tune learning rate and weight decay in DP training as in the regular training.

Theorem 1. Non-adaptive R-dependent automatic DP-SGD is equivalent to R-independent automatic DP optimizers.

Theorem 2. Adaptive R-dependent automatic DP optimizers (including AdaGrad, Adam, etc.) are equivalent to R-independent automatic DP optimizers.
